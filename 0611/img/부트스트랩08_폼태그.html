부트스트랩08_폼태그

다양한 폼 사용을 위한 폼 컨트롤 스타일, 레이아웃 옵션, 사용자 정의 컴포넌트의 예시와 사용가이드

폼태그 사용시 : form, fieldset, legend

폼태그 그룹제목

패스워드는 특수문자를 포함한 8-10까지 가능



폼 입력 양식의 크기 조정: .form-control-lg, .form-control-sm

.form-text : block 요소와 간격을 쉽게 맞출 수 있도록 상단여백이 추가. small, span 태그를 사용하면 좋다.



버튼 비활성화 -disabled

disabled 불리언 속성을 추가하면 비활성화 외형이 되어 포인터 이벤트가 삭제되면서 포커싱 되지 않는다.



읽기전용 - readonly

readonly 불리언 속성을 추가하여 입력값의 변경을 막을 수 있다. disabled와 달리 readonly는 계속 포커싱 및 선택이 가능하다.



라디오버튼

input: form-check-input // label : form-check-label



체크박스

input: form-check-input // label : form-check-label



선택목록 - select, option

.form-select 서식 적용, 크기 조정시에는 form-select-lg, form-select-sm을 사용한다



.form-select 하면 여러개의 select 항목이 보인다.multiple하면 ctrl이나 shift 누르면서 선택이 가능하다
