table class 기본 서식

  • class="table" : 테이블 기본 서식
  • class="table-striped" : 행마다 다른 색상 적용
  • class="table-bordered" : 테이블 테두리
  • class="table-hover" : 마우스 오버시 색깔 변하게
  • class="table-dark" : 어두워짐
  • class="table-primary", class="table-danger", class="table-success", class="table-info"
  • class="table-responsive{-sm|-md|-lg|-xl}" : 반응형 - 화면의 너비가 768미만일 때 가로로 스크롤 됨
  • class="text-center" : 가운데 정렬
