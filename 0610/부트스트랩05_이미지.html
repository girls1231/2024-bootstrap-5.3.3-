부트스트랩05_이미지

이미지태그를 사용하여 이미지를 삽입하고 다양하게 클래스를 적용해본다



1. 모서리 살짝 둥글게 - css(border-radius)

부트스트랩에서는 class="rounded"로 적용한다



2. 모서리 원으로 - css(border-radius : 50%)

부트스트랩에서는 class="rounded-circle"로 적용한다



3. 썸네일 이미지처럼

부트스트랩에서는 class="img-thumbnail"로 적용한다. border, padding, border-radius가 모두 적용됨



이미지와 문자를 사용하여 figure, figcaption 사용할 경우

Images in Bootstrap are made responsive with .img-fluid. max-width: 100%; and height: auto; are applied to the image so that it scales with the parent element.
